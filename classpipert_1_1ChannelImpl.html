Detailed Description

(Part of internal implementation.

)

This is the implementation of the Channel logic. This template free code has all necessary data extracted from the templated public interface. Every kind of Channel object has a ChannelImpl backing it up.

For an overview of the packet processing workflow, see the following diagrams.
