Detailed Description

template<class T>
class pipert::SenderChannel< T >

A Channel whose packets are automatically scheduled to be sent to the given remote computer.

SenderChannel is a special type of SchedulerChannel, where the Packets from the channel's buffer are automatically scheduled. The function to be called when processing data is always the SendPacket() function.
