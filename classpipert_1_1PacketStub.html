Detailed Description

template<class T>
class pipert::PacketStub< T >

A RAII object connecting a Packet and its containing Channel.

This object points to the connected Packet and Channel, so the connected objects are never moved when this stub is passed around. Submitting this as a parameter or returning as a value is supported through the implemented move operations.
