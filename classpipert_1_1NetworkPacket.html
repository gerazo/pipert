Detailed Description

template<class T>
class pipert::NetworkPacket< T >

Represents data which is transmitted between computers for processing.

NetworkPacket is a special type of Packet, which is used for pipeline communication between separated computers on the network. See Packet class for details.

Template Parameters
TType of user data which this packet will contain. Type must be Trivially Copyable to be appropriate for network communication.
